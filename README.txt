KALKULATOR SUHU
Program kalkulator suhu yang dibangun dengan Bash dan Python

========================================
CARA MENJALANKAN PROGRAM KALKULATOR SUHU
========================================

Ikuti langkah-langkah berikut untuk mulai menjalankan program. 

1. Buka Terminal dan masuk ke dalam folder hasil ekstrak program ini dengan:

cd 25.11.6401_kalkulator-suhu

2. Berikan izin eksekusi (execute permission) ke semua file dengan perintah berikut:

chmod +x *

3. Jalankan program utamanya dengan:

./25.11.6401_kalkulator-suhu.sh

(Catatan: Program ini harus dijalankan melalui file Bash (.sh) di atas, karena file Python yang ada hanya bertindak sebagai otak/logika di latar belakang).
